# MindMap
Mind mapping program written in Python with Tkinter

Python version used: 2.7

OS used: x86_64 GNU/Linux

Libraries needed:
    - Tkinter
    - tkMessageBox
    - PIL
    - threading
    - subprocess
    - ImageTk
    - random
    - time
    - datetime
    - parsedatetime
    - sys
    - json

To run program:
    - python mindMap.py

To run wrapper:
    - python wrapper.py
